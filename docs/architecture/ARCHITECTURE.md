## Why This Architecture?

### âœ… Benefits

1. **Testability**
   - Mock repositories easily
   - Test use cases without DB
   - Test controllers without HTTP

2. **Flexibility**
   - Swap SQLite â†’ PostgreSQL (just change repository)
   - Swap Fastify â†’ Express (just change HTTP layer)
   - Add GraphQL without touching business logic

3. **Maintainability**
   - Clear separation of concerns
   - Each layer has single responsibility
   - Easy to understand and modify

4. **Scalability**
   - Add new features without breaking existing code
   - Each use case is independent
   - Easy to split into microservices later
